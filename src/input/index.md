## API

### props

|   参数   |          类型          | 默认值 |      描述      |
| :------: | :--------------------: | :----: | :------------: |
| disabled |        boolean         | false  |    是否禁用    |
|   size   |      'lg' \| 'sm'      |   -    | 输入框尺寸大小 |
|  prefix  | string \| ReactElement |   -    | 输入框前端样式 |
|  suffix  | string \| ReactElement |   -    | 输入框后端样式 |
